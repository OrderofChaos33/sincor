Handoff contract (atomic):

4) Task market (faster, fairer, harder to game)

Use a sealed-bid second-price (Vickrey) auction per work unit with a quality-aware score. Orchestrator emits AuctionSpec; MARKET runs the auction; OPS enforces limits.

Bid payload:

{
  "task_id": "TT-...",
  "agent_id": "P7",
  "eta_s": 900,
  "ask": 24.0,
  "confidence": 0.86,
  "risk_adj": 0.04,
  "slot": 1,
  "explain": "Have existing template + brand pack cached"
}


Scoring (maximize):

value = V * confidence * freshness
cost  = ask + spinup_penalty + risk_cost
diversity_bonus = λ * (1 - jaccard(skill_vec(agent), skill_vec(last_k_winners)))
score = value - cost + diversity_bonus


V is the unit value from ORCH (derived from priority, deadline tightness, downstream dependencies).

freshness penalizes stale caches or outdated models.

spinup_penalty if agent currently cold.

risk_cost from OPS policy map (e.g., external write).

λ 0.05–0.15 to avoid monoculture and encourage exploration.

Winner selection: highest score wins; pay second price (next best score mapped back to cost).
Guards: per-guild spend caps; per-agent pacing; collusion detector = correlated bid patterns > τ over 24h → shadow audits.

Budget adherence: soft cap enforces VCG-like throttling; if second price > soft, ORCH may down-scope or re-decompose.

7) Orchestrator logic (pseudo)
def decompose(task):
    graph = plan(task)  # DAG of lots
    for lot in topo_sort(graph):
        spec = make_auction_spec(lot)
        bids = collect_bids(spec, T=spec.T_bidding_s)
        winner = vickrey_select(bids, diversity_lambda=spec.diversity_lambda)
        assign(lot, winner.agent_id, pay=winner.second_price)
